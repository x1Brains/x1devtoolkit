# X1 DevToolkit

Developer reference & toolkit for the X1 blockchain. Deployed on Vercel.

## Features

- **Network Toggle** — Switch between mainnet/testnet, auto-updates all RPC & explorer links
- **API Reference** — Every endpoint, mint address, program ID in one searchable page
- **Code Snippets** — Copy-paste TypeScript & Rust snippets for common operations
- **Program Workspace** — Anchor/Rust program management (coming soon)

## Setup

```bash
npm install
npm run dev
```

Opens on `http://localhost:5174`

## Deploy

Connected to Vercel via GitHub — push to `main` to deploy.
